WEB이란 무엇인가?

The World Wide Web(WEB), is a way of accessing information over the medium of the internet. It is an information-sharing model that is built on top of the internet. The web uses the HTTP protocol, only one of the languages spoken over the internet, to transmit data. Web services, which use HTTP to allow applications to communicate in order to exchange business logic, use the the web to share information. The web also utilizes browsers, such as Internet Explorer or Firefox, to access Web documents called webpages that are linked to each other via hyperlinks. Web documents also contain graphics, sounds, text and video.
